Report writers-room runs.

Arguments:

$ARGUMENTS

Steps:

1. Run `npm run room -- report $ARGUMENTS`.
2. For the relevant run, inspect `ROOM_REPORT.md`, `decision-log.md`, `idea-cards.jsonl`,
   and any `output/beat-board.md` or `output/table-read-checklist.md`.
3. Summarize selected, parked, rejected, and still-risky directions.
4. Do not draft from raw room output. Convert accepted direction into section
   contracts, issues, continuity notes, or a deliberate Chorus/drafting pass.
